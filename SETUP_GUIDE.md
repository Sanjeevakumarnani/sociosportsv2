# SocioSports Setup Guide

## 1. Automated (Recommended)
Run the PowerShell script in the root directory:
```powershell
.\setup-local.ps1
```

## 2. Start Servers
Run the following command from the root:
```bash
npm run dev
```

## 3. Access
- **Website**: [http://localhost:5173](http://localhost:5173)
- **Admin**: [http://localhost:5173/admin](http://localhost:5173/admin)
- **Admin Login**: `admin@sociosports.com` / `Admin@2026`

For detailed manual instructions and troubleshooting, see [local-setup.md](file:///c:/Users/ssk/Desktop/kimi/SocioSports/local-setup.md).
