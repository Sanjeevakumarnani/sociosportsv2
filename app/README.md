# SocioSports - India's Sports Ecosystem Platform

SocioSports is a high-performance sports management and networking platform designed for athletes, coaches, and organizers across India. This repository contains the professional React-based website with advanced animations and interactive features.

## 🚀 Key Features

- **Interactive Navigation:** Eye-tracking emoji header that follows cursor movement.
- **Dynamic Hero Section:** Auto-rotating action shots and feature highlights.
- **Sports on Wheels:** Integrated booking for mobile sports infrastructure and expert coaching.
- **Athlete/Coach Ecosystem:** Dedicated networking and growth tools for sports professionals.
- **Theme System:** Three premium design variants (Deep Navy, Royal Gold, Forest Green).
- **Modern Tech Stack:** Built with React, TypeScript, Vite, and GSAP.

## 🛠️ Technology Stack

- **Frontend:** React 18, TypeScript, Tailwind CSS
- **Animations:** GSAP (ScrollTrigger), Framer Motion
- **Icons:** Lucide React
- **Build Tool:** Vite

## 📦 Getting Started

### Prerequisites

- Node.js (v18.0 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
